MoonShotParsing
======



### Requirements:

- python >= 3.7

- Install all packages in `requirements.txt`

- Twitter developer account: config `API_KEY` and `API_KEY_SECRET` in 'config.py'

- Firebase certifiate file: config `CERTIFICATE_FILE` in 'config.py'

  

### Running

```bash
python -m app.main
```
